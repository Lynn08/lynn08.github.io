欢迎大家访问我的博客，此博客用于post日常遇到的知识点及bug.

Quick Start

在Lightning Component的开发中遇到这样的问题:
在Case对象上创建的QuickAction都在整个页面的右下角(Chatter部分)，这个是salesforce设定的布局，
这样就会导致我的页面会变得特别小，甚至不能够完全添加我们需要的字段，但是在今天看文档的时候，发现
这个问题是可以解决的，也就是说我们可以任意修改页面的布局(Record Page)，下面就来教大家解决这个问题。
